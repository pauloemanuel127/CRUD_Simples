Nesse projeto decidi criar minhas proprias funções para entender melhor sobre modularização e um pouco de recursão.

## Como utilizar

### No Windows:

1. instale o repositório na sua maquina
2. abra o terminal
3. siga a seguinte sequencia de comandos

```cmd
cd Downloads/CRUD_Simples
python3 main.py
```

### No Linux:

1. instale o repositório na sua maquina
2. abra o terminal
3. siga a seguinte sequencia de comandos

```bash
cd Downloads/CRUD_Simples
python3 main.py
```
