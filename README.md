# Sistema de CRUD simples usando lista encadeada

> Nesse projeto, usei os conhecimentos adquiridos por meio do livro *Entendo Algoritmos*, onde aprendi os conceitos de lista encadeada e decidi aplicá-los na prática para melhorar meu aprendizado.

Por meio desse projeto, consegui entender como é um CRUD e seu funcionamento. Além disso, pude praticar o desenvolvimento das minhas próprias funções e a criação de documentação.

## Listas Encadeadas

Fiz a lista encadeada usando dicionários no Python e referências de ponteiros nativos, sem auxílio de listas ou métodos prontos da linguagem.

## Funções

Nesse projeto, decidi criar minhas próprias funções para entender melhor a modularização. Além disso, pude praticar a criação de documentação de funções por meio de docstrings.

As funções do código utilizam a lógica de busca linear com complexidade de tempo O(n) para navegar entre os nós.

## Main

No arquivo main.py, tentei criar uma interação com o usuário pelo terminal que fosse fácil e acessível para qualquer usuário. O menu possui um sistema de validação e sanitização de dados com blocos try/except.

## Tecnologias e Conceitos

- **Linguagem:** Python 3.14
- **Estrutura de Dados:** Lista Encadeada Pura (simulada com dicionários)
- **Conceitos:** Modularização, Busca Linear, Complexidade de Tempo O(n), Tratamento de Exceções (try/except)

### No Windows, Mac ou Linux:

1. Instale o repositório na sua máquina
