# Sistema de CRUD simples usando lista encadeada

> nesse projeto usei meus conhecimentos adquiridos por meio do livro *Entendo Algoritmos*, onde aprendi os conceitos de lista encadeada e decidi aplicar na pratica para melhorar meu aprendizado.

Por meio desse projeto consegui entender como é um CRUD e seu funcionamento, além disso pude praticar desenvolver minhas proprias funções e gerar a documentação.

## Listas Encadeadas

Fiz a lista encadeada usando dicionários no Python e referências de ponteiros nativos, sem auxilio de listas ou metodos prontos da linguagem.

## Funções

Nesse projeto decidi criar minhas próprias funções para entender melhor sobre modularização e um pouco de recursão. Além disso pude ter prática também em criar documentação de funções por meio das docstrings.

As funções do código utilizam a lógica de busca linear com complexidade de tempo O(n) para navegar entre os nós.

## Main

No arquivo main.py tentei criar uma interação com o usuário pelo terminal que fosse fácil e acessivel para qualquer usuário. O menu possui um sistema de validação e sanitização de dados com blocos try/except.

## Tecnologias e Conceitos

- **Linguagem:** Python 3.14.5
- **Estrutura de Dados:** Lista Encadeada Pura (simulada com dicionários)
- **Conceitos:** Modularização, Busca Linear, Complexidade de Tempo O(n), Tratamento de Exceções (try/except)

## Como utilizar

### No Windows, Mac ou Linux:

1. instale o repositório na sua maquina
2. abra o terminal
